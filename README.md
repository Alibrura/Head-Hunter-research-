## 📌 Описание проекта

В рамках учебного проекта решается задача анализа данных о вакансиях для последующего создания модели машинного обучения, которая будет рекомендовать подходящие вакансии клиентам кадрового агентства, претендующим на позицию **Data Scientist**.

Проект демонстрирует навыки взаимодействия с реляционной базой данных PostgreSQL средствами Python (библиотека `psycopg2`), выполнения SQL-запросов и последующего анализа полученных данных.

**Бизнес-задача:** Провести исследование рынка вакансий в сфере Data Science для понимания структуры данных, выявления ключевых закономерностей и подготовки основы для рекомендательной системы.

## 🧱 Этапы исследования

### 1. Знакомство с данными (Data Understanding)
- Подключение к базе данных через `psycopg2`
- Изучение структуры таблиц, типов данных, первичных и внешних ключей

### 2. Предварительный анализ данных (EDA)
- Количество вакансий в базе данных
- Количество работодателей
- Количество регионов
- Количество сфер деятельности в базе данных

### 3. Детальный анализ вакансий
- Количество вакансий в каждом регионе по убыванию
- Средние значения для нижней и верхней границы зарплатной вилки
- Частотная таблица в зависимости от типа рабочего графика и типа трудосутройства

### 4. Анализ работодателей
- Топ-работодатели по количеству вакансий
- Распределение вакансий по сферам деятельности
- Крупнейшие компании-монополисты (Яндекс, Газпром и др.) – их доля на рынке

### 5. Предметный анализ (фокус на Data Science)
- Отбор вакансий, связанных с Data Science
- Сравнение зарплатных вилок в DS и других IT-направлениях
- Регионы с наибольшим спросом на DS-специалистов

## 💡 Выводы и заключение

Подводя итоги проведённого исследования, можно выделить основные этапы проделанной работы:

- ознакомление с данными (Data Understanding);
- предварительный анализ данных (EDA);
- детальный анализ вакансий;
- анализ работодателей;
- предметный анализ данных.

Все эти этапы являются неотъемлемой частью решения поставленной задачи — создания модели машинного обучения, которая будет рекомендовать вакансии клиентам агентства.

В рамках данного кейса мы исследовали вакансии в сфере Data Science. Очевидно, что при рассмотрении других профессиональных областей можно обнаружить иные закономерности, а возможно, и аномалии в данных.

Большинство данных, проанализированных в проекте, не содержат неожиданных или универсальных отклонений и в целом соответствуют реальной ситуации на рынке труда. Тем не менее были замечены потенциальные выбросы: например, в некоторых регионах не оказалось ни одной вакансии. Возникает вопрос: может ли в России быть регион с нулевым числом вакансий? Это требует дополнительной проверки данных и более глубокого анализа.

Обращает на себя внимание тот факт, что крупные компании-монополисты, такие как **Яндекс** и **Газпром**, представлены значительным количеством вакансий. В дальнейшем, при разработке модели машинного обучения, целесообразно учитывать предпочтения соискателей: многие люди стремятся попасть именно в крупные организации, где зачастую предлагаются конкурентная заработная плата и комфортные условия труда. Поэтому, если кандидату по всем параметрам подходит вакансия из крупной компании, такая вакансия должна рекомендоваться в первую очередь.

В ходе исследования были изучены различные характеристики вакансий с целью выявления значимых закономерностей — это необходимо для успешного решения поставленной задачи. Одним из ключевых факторов, влияющих на отклик соискателя, является уровень предлагаемой заработной платы (или зарплатная вилка). В связи с этим в нашей работе мы уделили особое внимание анализу заработных плат.

### Дополнительные результаты
- Рассмотрены 10 самых высокооплачиваемых вакансий в выбранном датасете.
- Исследованы 10 работодателей с самыми высокими средними зарплатами (с учётом всех их вакансий).
